Instructions for Form Fields

First Name -
  • Can start with a capital letter
  • Can have one or more words
  • Atleast two letters long
 Last Name -
  • Only Alphabets
  • Cannot have more than one word
 User Id -
  • Can start with a capital letter
  • Cannot have any whitespaces
  • Atleast 5 letters long
 Password -
  • 6 to 16 valid characters
  • Atleast one number
  • Atleast one special charracter
